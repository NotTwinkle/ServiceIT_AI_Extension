# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

These icons should use the Service IT Plus branding:
- Navy Blue: #002b5c
- Orange: #ff9900

You can create these using any image editing tool or online icon generator.

For now, you can use placeholder icons or generate them at:
- https://www.favicon-generator.org/
- https://www.canva.com/

**Tip:** Use a simple design with the Service IT Plus colors (navy background with orange chat icon).
